KBQA-R1: Relation Retrieval & Confidence Gating

RRCG：用稠密检索验证 agent 提出的关系 r_agent 与 KB schema 的匹配度

计算 r_agent 与当前实体所有邻居 schema 关系的 Sim，取 s_max，按阈值 tau_high=0.95 和 tau_low=0.3 分三档：

Auto-Validation

s_max >= 0.95

高置信匹配

自动替换为最佳匹配关系 r_s* 并执行

Tentative Accept

0.3 <= s_max < 0.95

暂时接受执行

返回 top-k 候选集及分数，附加不确定性提示

Rejection

s_max < 0.3

拒绝执行

返回完整邻居关系列表和分数的诊断信息

作用：将 LLM 的 schema 幻觉控制在单步级别——高置信自动修正，中置信提示验证，低置信强制重选
