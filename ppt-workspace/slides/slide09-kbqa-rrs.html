KBQA-R1: Referenced Rejection Sampling (RRS)

Cold Start 问题：标准 RS 接受率仅约 39%（需生成合法 S-Expression + schema 一致的关系 + 多跳路径），RRS 通过 gold 动作引导提升至 51%-67%

RRS 四步流程（用 Qwen-2.5-72B 生成，蒸馏到 Llama-3.1-8B）

1

Action Extraction：从 gold S-Expression 解析出 ground-truth 动作序列

2

Referenced Rollout：每步 prompt 包含下一个 GT 动作作为参考，生成推理轨迹

3

Trajectory Filtering：保留 F1(预测, gold) >= tau 且格式正确的轨迹

4

Reference Stripping：去除所有参考提示，SFT 只在 think/action token 上计算 loss

核心：模型被迫解释"为什么"每个动作正确（而非事后编造），且通过观察真实 KB 反馈内化动作-观察对应关系
