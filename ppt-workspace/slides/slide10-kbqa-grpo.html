KBQA-R1: RL 优化 (GRPO)

奖励设计

r_outcome：预测答案与 gold 的 F1 Score

r_format：tag 完整性和正确顺序

仅在 r_outcome > 0 时生效，防止"格式对答案错"获奖

R = 1.0*r_outcome + 0.1*I[r_outcome>0]*r_format

GRPO Advantage

每个 prompt 采样 n=5 条轨迹

A_i = r_i - mean(r_1...r_n)

无需学习 value network，低方差

vs KBQA-o1 (MCTS)

推理效率：

KBQA-o1: 28.8-78.0 次 LLM 调用/问题

KBQA-R1: 2.65-3.16 次（减少 80%+）

吞吐量：

GrailQA: 155.6 vs 5.9 问题/分钟（26x）

架构：

无需独立 reward model

无需测试时搜索

单一策略端到端
